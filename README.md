# 🎫 IT Support Ticket Analysis Dashboard

> An interactive Power BI dashboard analyzing 3,000+ IT support tickets across multiple products, years, and support engineers.

![Dashboard Preview](dashboard_preview.png)

---

## 📊 Project Overview

This project analyzes IT support ticket data from **2023 to 2026** to uncover trends in ticket volume, product-wise issues, and team performance. Built using **Power BI**, this dashboard helps support managers and stakeholders make data-driven decisions to improve response efficiency and resource allocation.

---

## 🔍 Key Insights

- 📌 **3,000+ total tickets** tracked across 4 years
- 📌 **2,000 support tickets** and **396 DEV tickets** identified and categorized
- 📌 **86 open tickets** flagged for immediate attention
- 📌 **EnSi** is the highest volume product with **1,293 issues**
- 📌 Ticket volume **peaked in 2025** and shows a sharp decline in 2026
- 📌 **Karthikeyan (37.32%)** handles the highest ticket load among support engineers

---

## 📈 Dashboard Features

| Visual | Description |
|---|---|
| 🔢 KPI Cards | Total, Support, DEV & Open ticket counts at a glance |
| 📊 Bar Chart | Total tickets trend by year (2023–2026) |
| 📉 Line Chart | Support ticket volume trend over time |
| 🍩 Donut Chart | Ticket distribution by product |
| 🥧 Pie Chart | Ticket count by Support Engineer |
| 📋 Table | Detailed product-wise issue/support count |
